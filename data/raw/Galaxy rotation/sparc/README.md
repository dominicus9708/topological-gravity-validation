# SPARC data

## Description
SPARC (Spitzer Photometry and Accurate Rotation Curves) is a database of 175 nearby disk galaxies with 3.6 μm photometry and high-quality HI/Hα rotation curves.

## Source
Official site: https://astroweb.case.edu/SPARC/

## Core reference
Lelli, F., McGaugh, S. S., & Schombert, J. M. (2016),
"SPARC: Mass Models for 175 Disk Galaxies with Spitzer Photometry and Accurate Rotation Curves",
The Astronomical Journal, 152, 157.
DOI: 10.3847/0004-6256/152/6/157
