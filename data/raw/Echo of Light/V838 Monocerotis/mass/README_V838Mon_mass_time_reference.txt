README_V838Mon_mass_time_reference.txt
======================================

Purpose
-------
This CSV collects mass-related or shell/ejecta-related literature values that can be
time-tagged or phase-tagged for V838 Mon.

Important caution
-----------------
V838 Mon is not a standard supernova case with a clean pre/post explosion mass budget.
Therefore, this table mixes:
- direct observational/event records,
- progenitor-model interpretations,
- secondary literature compilations of ejecta/envelope mass estimates.

Recommended usage
-----------------
1. Use the CSV as an auxiliary reference table.
2. Keep direct observational/event markers separate from later model-dependent mass estimates.
3. Do not identify the 2002-2004 light-echo dust automatically with the 2002 ejected material.
4. Prefer direct-paper rows when possible; use secondary-compilation rows as broad bounds only.

Column notes
------------
- time_phase: event-phase label
- approx_date_or_period: practical time anchor
- parameter_name: the quantity or event marker
- value_min/value_max: numeric bounds if available
- source_kind: whether the row comes from a direct paper statement or a later literature compilation
- note: short explanation of how to interpret the row
