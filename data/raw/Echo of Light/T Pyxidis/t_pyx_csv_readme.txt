T Pyxidis CSV package
Files:
- t_pyx_source_registry.csv : paper/source registry
- t_pyx_image_links.csv : public image links and image dates
- t_pyx_timeseries_extracted.csv : directly usable timeline seed
- t_pyx_mass_volume_summary.csv : mass/size/volume-related values and proxies

Important:
- Some values are direct literature values.
- Some are indirect computed proxies made from direct literature values.
- Volume entries are proxies only unless a paper gives an explicit 3D volume.
- The light echo structure is described as ring + shell, so spherical volumes should be used only as modeling proxies.
