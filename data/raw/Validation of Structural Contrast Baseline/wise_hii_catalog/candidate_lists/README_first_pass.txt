WISE H II bootstrap candidate notes
===================================

Recommended first-pass visual selection rule:
1. Start from wise_hii_shortlist_known_group_simple_shell.csv
2. Prefer objects with a single, clean ring/shell morphology
3. Avoid obviously overlapping complexes in the first test
4. Avoid highly irregular fragmented regions in the first test
5. Keep 3 final candidates for manual inspection

Suggested first validation goal:
- Build a radial profile from one clean shell-like target
- Define outer background ring
- Check whether sigma separates shell region from background
