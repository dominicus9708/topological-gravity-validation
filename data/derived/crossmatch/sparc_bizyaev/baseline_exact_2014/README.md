# Baseline exact-match structural subset (Bizyaev 2014)

This folder stores the conservative baseline subset obtained
from exact catalog matching between:

- SPARC galaxy index
- Bizyaev et al. (2014) edge-on structural catalog

Purpose:
- high-confidence structural crossmatch baseline
- preserved before incorporating additional edge-on catalogs
  (e.g. Bizyaev 2002, Kautsch 2006)

Contents:
- sparc_bizyaev_crossmatch_exact.csv
- sparc_bizyaev_crossmatch_matched.csv

Status:
- baseline preserved
- do not overwrite
