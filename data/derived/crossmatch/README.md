# Derived crossmatch products

## Description
This folder contains derived crossmatch tables generated within this project.
These files are not original survey products.

## Parent data sources
- SPARC
- DiskMass Survey
- S4G-related naming/structure support tables

## Reproducibility
Each derived file should be reproducible from:
1. raw survey files
2. preprocessing scripts
3. crossmatch scripts in `src/crossmatch/`

## Important note
When citing scientific use, cite the original survey papers in addition to this repository.
