README_SN2014J_pipeline.txt
===========================

Purpose
-------
This script uses the processed SN 2014J input CSV files under:

    data/derived/Echo of Light/input/SN 2014J

and writes results under:

    results/Echo of Light/output/SN 2014J/YYYYMMDD_HHMMSS

Processed input files created together with this script
------------------------------------------------------
- SN2014J_epochs_processed.csv
- SN2014J_measurements_processed.csv
- SN2014J_model_input_radius.csv
- SN2014J_auxiliary_processed.csv
- SN2014J_sources_combined.csv

Interpretation note
-------------------
This is a first standard-reference pipeline skeleton.
It is not yet a full topological or mass pipeline.
