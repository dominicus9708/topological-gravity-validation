Our galaxy Halo Stellar Kinematics - topological stage 011

Changes in this revision
- Robust structural aggregation is retained.
- Standard comparison is read from standard shell summary model files.
- Shell-level standard stream-overlap diagnostics are also ingested from standard stage 004.
- A shell-total structural contrast is defined as Sigma_shell_total = sigma_local_mean_shell + Sigma_bg_shell.
- The main topological gradient diagnostic is now computed from Sigma_shell_total, not only Sigma_bg_shell.
- shell_caution_label is added to distinguish sparse / stream-rich / nominal shells.
- Coupling remains diagnostic only and is not applied to prediction.
